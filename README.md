# Fundamentos de Testes - RPV 02

## Cenários de Testes

🛒 E-commerce (Cálculo de Frete e Cupom)Regra de Negócio: O frete é R$ 20,00 fixo. Se a compra passar de R$ 300,00, o frete é grátis. Cupons de 10% não se aplicam ao valor do frete.

👤 Cenário 2: Cadastro de Usuário (Validação de Dados)
Regra de Negócio: O nome deve ter pelo menos 3 caracteres. A idade deve ser entre 18 e 120 anos. A senha deve ter no mínimo 8 caracteres.

🏦 Cenário 3: Sistema Bancário (Transferência Pix)
Regra de Negócio: O saldo não pode ficar negativo (não há cheque especial). O limite por transação é de R$ 5.000,00.
